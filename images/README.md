## 画像の仕様

- フォーマット: PNG（配置済み）
- 解像度: 72dpi以上
- カラーモード: RGB
- 最適化: Web用に最適化（ファイルサイズを抑える）

## 代替画像の生成

画像が用意できない場合は、以下のようなプレースホルダーサービスを利用できます：
- https://placeholder.com/
- https://via.placeholder.com/

例: `<img src="https://via.placeholder.com/1920x1080/fef7f0/e67e22?text=Mika+Sato" alt="Mika Sato">`
